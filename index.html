Full-Stack Web Developer

Umumnya, seorang full stack developer diminta untuk menulis kode back-end, mengintegrasikan database, dan menulis HTML/CSS di bagian front-end yang terintegrasi dengan kode back-end tadi. Sebagai teknisi profesional, seorang full stack developer tentunya diharuskan memiliki kemampuan yang beragam dalam bahasa pemrogaman.

Front-End Web Developer

Seorang front end developer harus memastikan tampilan website tersebut telah sesuai atau belum dengan desain awal dan tepat dalam hal fungsionalitasnya. Untuk tampilan yang berada pada website, aplikasi, menu, button, animasi, dan interaksi yang dapat disentuh oleh user merupakan tugas dari seorang front end developer.
